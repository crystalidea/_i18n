AnyToISO is able to create BIN/CUE image from CD, Audio CD or DVD. Just select "CD/DVD disk to ISO" tab, select the drive and check the "Create BIN/CUE image" option.

More information: BIN/CUE images are CD(DVD) images that come in both a .bin and .cue file. The .cue file (cue-sheet) contains the track layout information and the .bin file holds the actual data (in raw mode). Also called BIN/CUE. BIN/CUE files with data can be converted ISO, Audio images can be extracted to .WAV files.
